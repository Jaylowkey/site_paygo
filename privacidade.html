1 Informações que Coletamos

Coletamos apenas as informações necessárias para fornecer nossos serviços de forma segura e eficiente.

Dados Pessoais:

  • Nome completo
  • Número de telefone (M-Pesa/e-Mola)
  • Endereço de email
  • Histórico de transações

Dados Técnicos:

  • Endereço IP
  • Tipo de dispositivo e navegador
  • Dados de cookies e analytics

2 Como Usamos suas Informações

Suas informações são utilizadas exclusivamente para:

Processar Pagamentos

Para completar suas transações internacionais

Comunicação

Para enviar confirmações e suporte

Segurança

Para prevenir fraudes e proteger sua conta

Melhorias

Para otimizar nossos serviços

3 Compartilhamento de Dados

Nunca Vendemos seus Dados

Seus dados pessoais nunca são vendidos, alugados ou compartilhados com terceiros para fins de marketing.

Podemos compartilhar dados apenas com:

  • Processadores de pagamento (M-Pesa, e-Mola) - apenas dados necessários
  • Autoridades legais - apenas quando exigido por lei
  • Prestadores de serviço - com acordos de confidencialidade

4 Segurança de Dados

Implementamos medidas de segurança robustas para proteger suas informações:

Criptografia SSL

Todos os dados são transmitidos de forma segura

Armazenamento Seguro

Dados guardados em servidores protegidos

Acesso Restrito

Apenas pessoal autorizado tem acesso

Actualizações

Sistemas de segurança sempre actualizados

5 Seus Direitos

Você tem os seguintes direitos sobre seus dados pessoais:

  • Acesso

    Solicitar uma cópia dos dados que temos sobre você

  • Correcção

    Solicitar correcção de dados incorrectos ou incompletos

  • Eliminação

    Solicitar eliminação dos seus dados (com ressalvas legais)

  • Oposição

    Opor-se ao processamento dos seus dados em certas circunstâncias

Para exercer estes direitos, contacte-nos via suporte@paygo.co.mz ou WhatsApp +258 83 752 2255.
